jQuery

Diff. between window.onload and $(function)

  • $(function) will execute immediately after the dom obj is ready
  • window.onload will execute after the dom is ready and all content is loaded. (eg. img, iframe tag)
  • when multiple window.onload is used, only the last one will execute
  • all $(function) will execute one by one

Event

  • activate when not passing para.
  • register when passing para.
    click()
    mouseover()
    mouseout()
    bind() register multiple events one() same as bind(), but the events only response once
    unbind()
    live() same as bind(), but the events also work for the elements after binding. see exp. below

Event Bubble

  • when two element is nested and register the same event, when the event is activate by the child element, it will also propagate to the parent element.

How to prevent bubbling

  • return false in the child function

Event Obj

  • Event obj will automatically create when an event is activated.

  • a lot of information about the event is store in the event obj.

  • How to access the event obj

  • use “event” as a para. in the event function

  • common use of event obj

  • when bind is used to bind mutilple evnets, event can be used to find out the activated event
