Posted on 2020-08-29 | In Server , Servlet

Filter

What is Filter

  • It is a interface under JavaEE
  • it can be used to intercept requests, filter response
  • common useage: Permission check, log, transcation management

Life cycle

  • contructor and init() start when web project starts
  • doFilter() exe when any request matches the url pattern
  • destroy() exe when web project ends

FilterConfig

  • FilterConfig obj can used to get filter-name, init-param and servletContext

    How to implement Filer

  • write a class implement Filter
  • impl doFilter()
  • config filter tag in web.xml

Example

FilterChain

  • the prority of different filter depends on the writting position in web.xml
  • multiplt filters use the same thread and request obj in default
