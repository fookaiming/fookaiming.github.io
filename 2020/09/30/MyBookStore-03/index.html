MyBookStore 03

Posted on 2020-09-30 | In Project , MybookStore

Objective

  1. change the static html page to jsp page, for dynamically show information from server
  • eg. show error message in login page
  1. combine the two servlet to one servlet
  2. change the if else to reflection
  3. imp an abstract BaseServlet for different Servlet class
  4. use BeanUtils to inject params to replace manully injection
  5. keep the user inputed infos in login page

    Workflow

    Objective 1

  6. add <%@ page contentType=”text/html;charset=UTF-8” language=”java” %> in the html
  7. change the .html to .jsp
  8. use idea (cmd + shift + r) to search .htm for entire directory
  9. change the static base tag to dynamic base tag
  10. In web layer, save the errMsg and user’s input in request attribute
  11. In jsp, show the errMsg and user’s input

    Objective 2

  12. add an hidden input in jsp
  13. create and impl the UserServlet class

    Objective 3

  14. getDeclaredMethod
  15. invoke

    Objective 5

  16. Create a WebUtils and imp the injectBean() method

    Technique

    Objective 1

  • save the infos in request attribute
  • in jsp, retirve the infos from request attribute

    Objective 2

  • use hidden input to send the “action” to server
  • server can determinate which method to call by the “action”

    Objective 3

    Objective 5

  • BeanUtils injects the obj by getter and setter
  • request properties can get by req.getParameterMap()

    Problem Encountered

  1. resource eg css, img not working
  • since base tag is linked to localhost, when user request the url it cannot get the resource in his own localhost
  1. java.lang.IllegalStateException: Cannot forward after response has been committe
  • this is beacuse no return statement after request redirection

    Solution

  1. change the base tag to dynamically link the resource
  2. add return statement after every redirection
