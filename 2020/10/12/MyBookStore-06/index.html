Posted on 2020-10-12 | In Project , MybookStore

Objective

  1. Implement login and logout function

  5. impl Order module

Workflow

Objective 1

  1. use cookie to save usename in login page
  2. display the last logined username
  3. show username in login_success page
  4. Logout

Objective 2

Objective 4

  3. display Cartitems in JSP

    Objective 5

  4. impl db table
  5. impl BookOrderDao, OrderItemDao

Problem Encountered

Objective 3

  • browser do not refresh the img due to the cache

Solution

Objective 3

  • this is due to the same request for same url
  • browser use cache instead of making real request
  • we can add a ramdom string in img src to prevent the cache problem
