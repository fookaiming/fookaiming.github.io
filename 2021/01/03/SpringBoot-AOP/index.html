What is AOP

  • a concept to implement feature for mutiple service

Spring AOP

  • spring aop is implement in runtime weaving(create proxy), it can only weave in method
  • Spring also support AspectJ(compilation weaving)

JDK Dynamic proxy

  • JDK Dynamic proxy can only proxy by interface
  • target class needs to implement an interface, which is then also implemented by the proxy class

CGLIB(use when class do no have interface)

  • CGLIB use dynamic Bytecode to create proxy by creating the subclass of the target class
