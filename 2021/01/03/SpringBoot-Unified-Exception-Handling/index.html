Unified Exception Handling

@ExceptionHandler

  • scope: method
  • this method will be call when exception occurs in any controller

    Example

    @ModelAttribute

  • scope: method
  • this method will be call before controller, it is used to inject obj to model when multiple request uses the same obj

Example

@DataBinder

  • scope: method
  • this method will be call before controller, it is used to convert request params. to different type.
  • eg: convert a username and password to a user obj, so controller use the user obj
